Project Overview

This project implements an advanced news classification system using GloVe embeddings and machine learning techniques. It leverages the power of word embeddings to understand the context and semantics of news articles, addressing the limitations of traditional methods like TF-IDF.

Skills: Java, HTML, CSS, GloVe, Machine Learning, Deeplearning4J, ND4J, Data Processing, Neural Networks
